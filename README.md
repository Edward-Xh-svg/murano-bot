# 🏛️ Malines Hostaka Stock Bot

بوت تيليجرام لإدارة الشركات في عالم Hostaka — محاكاة اقتصادية للقرن العشرين.

---

## 📁 هيكل الملفات

```
hostaka_bot/
├── bot.py              # الملف الرئيسي للبوت
├── config.py           # الإعدادات (التوكن، ID الأدمن، ...)
├── database.py         # قاعدة البيانات SQLite
├── ai_analyzer.py      # تحليل الشركات بالذكاء الاصطناعي
├── scheduler.py        # التقارير اليومية التلقائية
├── admin_add_secret.py # أداة إضافة المعرفات السرية
└── requirements.txt    # المكتبات المطلوبة
```

---

## ⚙️ طريقة التشغيل

### 1. تثبيت المكتبات
```bash
pip install -r requirements.txt
```

### 2. إعداد config.py
افتح `config.py` وأضف:
- `BOT_TOKEN` — توكن البوت من BotFather
- `ADMIN_ID` — ID حسابك في تيليجرام
- `GROUP_ID` — ID المجموعة لنشر الشركات (اختياري)
- `CLAUDE_API_KEY` — مفتاح Claude AI API

### 3. إضافة المعرفات السرية
قبل استقبال أي طلبات، أضف المعرفات السرية للجهات:
```bash
python admin_add_secret.py
```

### 4. تشغيل البوت
```bash
python bot.py
```

---

## 🎮 كيفية عمل البوت

1. **المستخدم** يضغط `/start` ثم "ابدأ"
2. يختار "إنشاء شركة" ويملأ الاستمارة خطوة بخطوة
3. البوت يتحقق من المعرف السري
4. الذكاء الاصطناعي يحلل الشركة ويقدّر الأرباح والمخاطر
5. يُرسل التقرير للأدمن للموافقة
6. الأدمن يرسل: `نعم_USER_ID` أو `لا_USER_ID`
7. عند القبول: تُنشر الشركة في المجموعة ويُبلَّغ المستخدم يومياً

---

## 👮 أوامر الأدمن

- `نعم_123456789` — قبول طلب المستخدم
- `لا_123456789` — رفض الطلب

---

## 📊 التقارير اليومية

كل 24 ساعة، يرسل البوت لكل مالك شركة تقريراً يتضمن:
- الأرباح/الخسائر المقدرة بعملة الشركة
- حالة السوق
- توصية يومية

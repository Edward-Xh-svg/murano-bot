# 🏛️ Malines Hostaka Stock Bot
بوت تيليجرام لإدارة الشركات في عالم Hostaka — محاكاة اقتصادية للقرن العشرين.
## 📁 هيكل الملفات
```text
hostaka_bot/
├── bot.py              # الملف الرئيسي للبوت
├── config.py           # الإعدادات (التوكن، ID الأدمن، ...)
├── database.py         # قاعدة البيانات SQLite
├── ai_analyzer.py      # تحليل الشركات بالذكاء الاصطناعي (Gemini 2.5 Flash)
├── scheduler.py        # التقارير اليومية التلقائية
├── admin_add_secret.py # أداة إضافة المعرفات السرية
└── requirements.txt    # المكتبات المطلوبة

```
## ⚙️ طريقة التشغيل
### 1. تثبيت المكتبات
قم بتثبيت المكتبات المطلوبة عبر الأمر التالي:
```bash
pip install -r requirements.txt

```
### 2. إعداد config.py
افتح config.py وأضف المتغيرات التالية:
 * BOT_TOKEN — توكن البوت الذي حصلت عليه من BotFather.
 * ADMIN_ID — ID حسابك في تيليجرام.
 * GROUP_ID — ID المجموعة لنشر الشركات (اختياري).
 * GEMINI_API_KEY — مفتاح Google AI API الخاص بك لاستخدام نموذج Gemini 2.5 Flash مجاناً.
### 3. إضافة المعرفات السرية
قبل استقبال أي طلبات، أضف المعرفات السرية للجهات من خلال هذا الأمر:
```bash
python admin_add_secret.py

```
### 4. تشغيل البوت
قم بتشغيل الملف الرئيسي للبوت:
```bash
python bot.py

```
## 🎮 كيفية عمل البوت
 1. **المستخدم** يضغط /start ثم "ابدأ".
 2. يختار "إنشاء شركة" ويملأ الاستمارة خطوة بخطوة.
 3. البوت يتحقق من المعرف السري في قاعدة البيانات.
 4. الذكاء الاصطناعي (**Gemini 2.5 Flash**) يحلل الشركة ويقدر الأرباح والمخاطر.
 5. يُرسل التقرير للأدمن للموافقة عليه.
 6. يرسل الأدمن أحد الأمرين: نعم_USER_ID أو لا_USER_ID.
 7. عند القبول: تُنشر الشركة في المجموعة ويُبلَّغ المستخدم يومياً بتقرير عن شركته.
## 👮 أوامر الأدمن
 * نعم_123456789 — قبول طلب المستخدم.
 * لا_123456789 — رفض الطلب.
## 📊 التقارير اليومية
كل 24 ساعة، يرسل البوت لكل مالك شركة تقريراً يتضمن:
 * الأرباح/الخسائر المقدرة بعملة الشركة.
 * حالة السوق.
 * توصية يومية.
